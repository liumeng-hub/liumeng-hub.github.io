文章除最后一部分其余来自https://zhuanlan.zhihu.com/p/430035973，链接已挂，此处用于备份

一、下载clash并解压（已完成此操作的可直接看第二步）

  • 官方网站 clash 下载clash-linux-amd64-v1.11.8.gz（也可以下载最新的版本，前缀是clash-linux-amd64即可）
  • 外网访问慢，使用到的下载资源可在baidu网盘下载

  • 进入下载的文件夹（我的在此文件夹，username为用户文件夹） ，在页面空白处右键，在终端打开

  • 解压

二、进入新建的这个Clash文件夹，下载config.yaml和Country.mmdb (注意，【订阅链接】是写自己在代理那里获取的）

三、启动clash（使用当前目录下的配置件）

（1）执行 ./clash -d . 即可启动 Clash，如提示权限不足，先执行 chmod +x clash ，再执行./clash -d . 出现如下，并保持此终端打开

（2）打开配置文件config.yaml ，给它设置一个密码： # RESTful API 的口令 secret: ‘123456’

（3）打开系统设置，选择网络，点击网络代理右边的 ⚙ 按钮，选择手动，填写 HTTP 和 HTTPS 代理为 127.0.0.1:7890，填写 Socks 主机为 127.0.0.1:7891，即可启用系统代理。

（4）访问 Clash Dashboard 可以切换节点、测延迟等操作。（这里的代理模式选规则）
