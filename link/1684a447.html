ubuntu系统软件安装

第一小节：安装ubuntu系统

  • 网页搜索ubuntu，下载iso镜像文件
  • 下载Rufus制作U盘启动盘
  • 磁盘分区，为ubuntu系统分配空间，单系统忽略
  • 进入bios后，选择U盘启动，根据提示安装
  • 当出现pcie报错时，在grup界面选中ubuntu，按e进行编译，在quiet salash后空格添加pcie_aspm=off
  • 安装完成后，重启时仍需要重复上一步骤，重启后在终端输入sudo gedit /etc/default/grub
  • 在文件中找到GRUB_CMDLINE_LINUX_DEFAULT，修改关键字在quiet salash后空格添加pcie_aspm=off
  • 保存退出，更新grubsudo update-grub

第二小节：安装pycharm

  • 在pycharm官网下载社区版

  • 解压文件，打开终端

  • 进入到pycharm对应文件的bin文件夹下

  • 执行 sh ./pycharm.sh 命令

  • 根据指引按步骤安装

  • 输入命令 sudo gedit ~/.bashrc 更改ubuntu配置文件

  • 最后加上alias pycharm="pycharm-community-2021.3/bin/pycharm.sh 路径"保存

  • source ~/.bashrc更新配置文件

  • crtl + alt + t 打开终端，输入pycharm打开软件

    PS：没有新建项目可能是窗口太小显示不全

第三小节：安装anaconda

  • 通过清华镜像源下载anaconda的发行版本Anaconda3-2019.10-Linux-x86_64.sh

  • 执行命令sh Anaconda3-2019.10-Linux-x86_64.sh

  • 根据指引完成安装

  • 终端中输入conda，如果没找到指令则进行下一步，

  • 执行echo 'export PATH="/home/用户名/anaconda3/bin:$PATH"' >> ~/.bashrc

  • source ~/.bashrc更新配置文件

  • 终端中输入conda，安装完成

  • 配置国内源

  • pip换源：pip config set global.index-url https://mirrors.aliyun.com/pypi/simple/

第四小节：安装labelme

  • pip install --upgrade setuptools
  • sudo pip install labelme -i https://pypi.tuna.tsinghua.edu.cn/simple some-package
  • 终端输入labelme，打开软件

第五小节：安装docker

方法一：curl下载

  • curl -sSL https://get.daocloud.io/docker | sh

方法二：yum下载

  • 配置yum源

# vim /etc/yum.repos.d/docker.repo

[dockerrepo]

name=Docker Repository

baseurl=https://yum.dockerproject.org/repo/main/centos/$releasever/

enabled=1

gpgcheck=1

gpgkey=https://yum.dockerproject.org/gpg

  • 通过yum安装

# yum install docker-engine

# service docker start

# service docker status

  • 日志

# vim /var/log/docker

第六小节：安装typora

直接应用商店下载即可

第七小节：ubuntu使用

1.解压rar文件

比如你有一个nam.rar的文件，

sudo unrar name.rar

但是一般会显示unrar命令不存在，那么你需要安装rar

sudo apt-get install rar

然后你在执行第一个命令，提示还是这一句：unrar命令不存在

那么你需要

sudo apt-get install unrar

那么你再用第一个，命令，需要带上参数：

sudo unrar x name.rar

sudo unrar e name.rar

其中x参数是解压到一个文件夹，e参数是把里面的文件都解压到当前路径下。

2.添加新用户

  • Ctrl+Alt+T打开终端，输入sudo adduser ubuntu(ubuntu是我创建的新用户名)，回车之后输入密码，根据提示输入新用户的密码，直接回车即可。

  • 输入sudo adduser ubuntt(新创建的用户名) sudo,给新用户赋予管理员的权限。

  • 切换到新用户，Ctrl+Alt+T打开终端，输入sudo su查看添加管理员权限是否成功。

3.监控gpu

  • 先使用sudo apt install nvidia-340命令安装
  • 然后nvidia-smi命令单次查看GPU使用率
  • 实时显示，可以使用nvidia-smi -l 1 命令每秒刷新一次。
